#  Student Score Predictor

##  About the Project

This project is a simple Python-based tool that predicts a student's marks based on the number of hours they study daily.

The idea behind this project is to understand how study time can affect academic performance using a basic prediction model.

---

##  Problem Statement

Many students are unsure about how much they need to study to achieve good marks. There is no simple way to estimate their performance based on effort.

---

##  Solution

This project uses a simple mathematical relationship between study hours and marks to give an approximate prediction. It also assigns a grade based on the predicted marks.

---

##  Features

* Takes study hours as input
* Predicts marks instantly
* Assigns grade (A, B, C, etc.)
* Handles invalid inputs
* Allows multiple predictions in one run

##  How to Run the Project

### Step 1: Install Python

Make sure Python is installed on your system.
You can check by running:
python --version

---

### Step 2: Download the Project

* Click on **Code → Download ZIP**
  OR
* Clone using Git:
  git clone https://github.com/your-username/student-score-predictor.git

---

### Step 3: Navigate to Folder

Open terminal or command prompt and go to the project folder:
cd student-score-predictor

---

### Step 4: Run the Program

Run the Python file:
python main.py

---

##  How to Use

1. Enter your daily study hours when prompted
2. The program will display:

   * Predicted marks
   * Expected grade
3. Type `exit` to stop the program
